Elios Language

Elios est un langage de script simple, compilé en C. L'idée c'est d'écrire du code lisible sans se prendre la tête — tout passe par §fonction[args], point.

v1.8.0 Production C GitHub

Pourquoi Elios ?

Syntaxe uniforme — toutes les fonctions suivent le même schéma
Variables globales partagées entre fichiers
Système de plugins extensible via ./addons
Mode debug intégré pour diagnostiquer facilement

Démarrage rapide

Tu es à deux commandes d'un programme qui tourne.

Exécuter un programme

elios programme.elios

Mode debug

elios programme.elios --debug

Hello World

§log[Bonjour le monde !]

Avec des variables

§var[nom;Alice]
§log[Bonjour $nom !]

Syntaxe de base

Tout fonctionne pareil — une fois que t'as compris le principe, tu sais tout écrire.

§fonction[arg1;arg2;arg3]

Règles clés

Toutes les fonctions commencent par §
Les arguments sont entre [...]
Plusieurs arguments séparés par ;
Variables référencées avec $NOM_VAR
Commentaires commencent par #

Exemple

# Ceci est un commentaire
§var[x;42]
§log[$x]

Entrée / Sortie

Fonctions pour afficher du texte et lire la saisie utilisateur.

§log[texte]
Affiche du texte dans la console.
§log[Bonjour le monde !]
§input[invite]
Lit la saisie de l'utilisateur.
§var[nom;§input[Quel est votre nom ?]]
§log[$nom]
§clear[]
Efface l'écran de la console.
§clear[]

Variables

Une variable définie quelque part est accessible partout — même dans un fichier chargé avec §require. Pas de portée compliquée.

§var[nom;valeur]
Définit ou met à jour une variable.
§var[age;25]
§var[nom;Alice]
§var[pi;3.14159]

Utiliser les variables

Référencez une variable avec $NOM_VARIABLE :

§var[salut;Bonjour]
§log[$salut le monde !]

Contrôle du flux

Conditions, boucles et instructions de saut.

Conditionnelle

§if[condition]
    §...code...
§elseif[condition]
    §...code...
§else
    §...code...
§endif
Exemple
§var[age;25]
§if[$age >= 18]
    §log[Adulte]
§elseif[$age >= 13]
    §log[Ado]
§else
    §log[Enfant]
§endif

Boucle while

§while[condition]
    §...code...
§endwhile
Exemple
§var[compte;0]
§while[$compte < 5]
    §log[$compte]
    §var[compte;§add[$compte;1]]
§endwhile

Boucle for

§for[variable;debut;fin]
    §...code...
§endfor
Exemple
§for[i;0;5]
    §log[Itération $i]
§endfor

If / Elseif / Else

§if[condition]
    §...code...
§elseif[condition]
    §...code...
§else
    §...code...
§endif

Sauts

§break[]
Quitter la boucle
§continue[]
Passer à l'itération suivante
§exit[code]
Quitter le programme avec un code de sortie
§exit[0]

Opérations sur fichiers

Lire, écrire et gérer des fichiers depuis Elios.

§getPath[relatif]
Résout un chemin relatif au répertoire courant.
§var[p;§getPath[data/fichier.txt]]
§log[$p]
§readFile[chemin]
Lit un fichier.
§var[contenu;§readFile[fichier.txt]]
§writeFile[chemin;contenu]
Écrit dans un fichier.
§writeFile[sortie.txt;Bonjour]
§isFileExist[chemin]
Vérifie l'existence du fichier.
§log[§isFileExist[fichier.txt]]
§createFile[chemin]
Crée un nouveau fichier.
§createFile[nouveau.txt]

Multi-fichiers

Tu peux répartir ton code sur plusieurs fichiers .elios et les assembler avec §require.

§require[chemin]
Inclut un fichier .elios.
§require[config.elios]
§require[lib/helpers.elios]

Fonctionnalités

Chargement avant exécution
Variables partagées entre fichiers
Imbrication des require
Détection des dépendances circulaires

Exemple

config.elios
§var[APP_NAME;MonApp]
§var[VERSION;1.8.0]
main.elios
§require[config.elios]
§log[Lancement de $APP_NAME v$VERSION]

Utilitaires

Fonctions de temps, de pause et de contrôle du programme.

§time[]
Retourne le timestamp Unix en millisecondes.
§var[ts;§time[]]
§log[$ts]
§date[format]
Retourne la date/heure selon le format. Tokens : YYYY MM DD HH mm ss
§log[§date[YYYY-MM-DD]]
§log[§date[HH:mm:ss]]
§sleep[ms]
Pause l'exécution pendant ms millisecondes.
§sleep[1000]
§log[1 seconde écoulée]
§exit[code]
Quitte le programme avec le code de sortie indiqué.
§exit[0]
